💡 TIPS:
═══════════════════════════════════════════════════════════════════════════

✓ If port 5000 is already in use:
  Edit app.py and change: app.run(debug=True, port=5001)

✓ To reset everything from scratch:
  python reset_db.py

✓ To view database with GUI:
  Download "DB Browser for SQLite" (sqlitebrowser.org)

✓ To backup your data:
  Copy instance/site.db to a safe location

✓ For production use:
  • Change SECRET_KEY to a secure random value
  • Switch to PostgreSQL database
  • Add HTTPS/SSL
  • Set debug=False
