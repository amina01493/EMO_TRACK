🔒 SECURITY NOTES
───────────────────────────────────────────────────────────────────────────

⚠️  DEVELOPMENT VERSION - NOT FOR PRODUCTION!

Before deploying:
  1. Change SECRET_KEY in app.py to a secure random string
  2. Set debug=False in app.py
  3. Use PostgreSQL instead of SQLite
  4. Add HTTPS/SSL certificates
  5. Add email verification for registration
  6. Implement password reset via email
  7. Add rate limiting for login
  8. Add CAPTCHA for registration
  9. Implement role-based access control
  10. Use environment variables for secrets
